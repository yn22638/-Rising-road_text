# 函数

>作用域：变量可以起作用的范围   分类：全局作用域、局部作用域
>全局变量和局部变量

```html
    全局变量
        在任何地方都可以访问到的变量是全局变量
    局部变量
        只在固定的代码片段内
```
### 作用域链
>只在函数可以制造作用域结构，那么只要是代码，就 至少有一个
* 简单来说，作用域可以用以下几句话来概括（确定一个变量来自哪个区域）
    1. 查看当前作用域，如果当前作用域声明了这个变量，就确定结果

### 变量提升
* 定义变量的时候，变量的声明会被提升到当前作用域的最上面，变量的赋值不会提升
